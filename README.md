# Medico Manager

Medico Manager is a web-based application for managing medical appointments, medicines, suppliers, and user/admin dashboards. This project is built using PHP and is intended to streamline operations for medical clinics or pharmacies.

## Features
- User and Admin Dashboards
- Appointment Management
- Medicine Inventory and Orders
- Supplier Management
- User Registration and Authentication
- Contact and About Us Pages

## Folder Structure
- `image/` - Contains image assets
- `.php files` - Core application pages and logic
- `medico_db (1).sql` - Database schema

## Getting Started
1. **Clone the repository** to your local machine.
2. **Set up the database** using the provided SQL file (`medico_db (1).sql`).
3. **Configure database connection** in `db_connect.php`.
4. **Run the application** on a local or remote PHP server.

## Requirements
- PHP 7.x or higher
- MySQL or compatible database
- Web server (Apache, Nginx, etc.)

## Usage
- Access the dashboard via `dashboard.php` for admin features.
- Users can sign up and log in via `signUp.php` and `index.php`.
- Manage medicines, appointments, and suppliers through respective pages.
